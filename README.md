# ⚡ Calculadora de Consumo de Energia

Uma calculadora simples em Python para estimar o consumo de energia elétrica de aparelhos domésticos e o custo aproximado por mês.

---

## 🎯 Objetivo do Sistema

O sistema permite que o usuário:

- Informe o nome do aparelho
- Informe a potência em Watts
- Informe o tempo médio de uso diário em horas

E receba como resultado:

- Consumo mensal em kWh
- Custo estimado do consumo

---

## 🖥 Linguagem Utilizada

- **Python** 🐍
